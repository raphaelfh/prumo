## Modelo de Licenciamento Duplo

Este projeto utiliza um modelo de **licenciamento duplo**:

- **AGPL-3.0 (Padrão)**: Todo o código está disponível sob a GNU Affero General Public License v3.0
- **Licença Comercial**: Disponível mediante pagamento para uso proprietário/comercial sem obrigações de copyleft

## Diretrizes para o AI

### Licenciamento

Este projeto utiliza **licenciamento duplo** (AGPL-3.0 + comercial). A informação de licenciamento está centralizada no arquivo `LICENSE` na raiz do repositório.

**Não é necessário** adicionar headers de copyright em arquivos individuais. Esta prática foi removida para:
- Reduzir overhead de manutenção
- Manter o código mais limpo
- Seguir práticas modernas de projetos open-source (2024-2025)
- O arquivo `LICENSE` na raiz é legalmente suficiente para proteger o projeto

### Contribuições e CLA

- Contribuições externas devem ser gerenciadas via **CLA Assistant** (configurado externamente)
- O AI não deve adicionar headers de copyright em novos arquivos

## Informações sobre AGPL-3.0

### O que é Copyleft?

A AGPL-3.0 é uma licença copyleft forte que garante:
- Liberdade de usar, modificar e distribuir
- **Obrigação**: Se você usar o software em uma aplicação web/SaaS, deve disponibilizar o código-fonte completo sob AGPL-3.0

### Licença Comercial

Empresas que desejam usar o software sem abrir seu código podem adquirir uma licença comercial que:
- Remove as obrigações de copyleft
- Permite uso proprietário
- Inclui suporte e garantias legais
